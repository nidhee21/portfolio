## 👩‍💻 About This Project

> *"Building a career at the intersection of code & security."*

This is my personal portfolio website — a terminal-themed, fully responsive site showcasing my skills, projects, certifications, and journey in Cybersecurity & Development.

Built from scratch with pure **HTML · CSS · JavaScript** — no frameworks, no templates.

---

## ✨ Features

- 🖥️ **Terminal-style hero** — animated CLI aesthetic with typewriter effect
- 📱 **Fully responsive** — works seamlessly on mobile, tablet, and desktop
- 🌙 **Dark theme** — sleek, cybersecurity-inspired design
- ⚡ **Zero dependencies** — pure HTML/CSS/JS, no bloat
- 📬 **Contact form** — with live feedback on submission
- 🎨 **Smooth animations** — scroll-triggered reveals and hover effects
